# AIチャットアプリケーション機能要件仕様書

## 1. システム概要
このAIチャットアプリケーションは、Pythonを使用して設計され、OpenAIのAPIを活用したリファクタリング提案や仕様書の自動生成を可能にするツールです。ユーザーは、コードのリファクタリング提案、詳細なプログラム仕様書の生成、機能要件仕様書の生成などを行うことができます。これにより、開発者はコード品質の向上やシステムの理解を深めることができます。

## 2. 主要機能要件
- **コードのリファクタリング提案**: `RefactoringChecker` クラスを使用して、コードのリファクタリング提案を生成し、保存します。
- **詳細仕様書の生成**: `DetailedSpecificationGenerator` クラスを用いて、既存の機能要件仕様書とソースコードを基に詳細なプログラム仕様書を生成し、保存します。
- **機能要件仕様書の生成**: `SpecificationGenerator` クラスを使用して、Pythonコードを解析し、機能要件仕様書を生成します。
- **Pythonファイルのマージ**: `PythonFileMerger` クラスを用いて、プロジェクト内のPythonファイルを一つのファイルにマージし、ディレクトリ構造を表示します。

## 3. 非機能要件
- **パフォーマンス**: 各機能は、ユーザーが待たされることなく迅速に応答できるように設計されています。
- **セキュリティ**: OpenAI APIキーは設定ファイルで安全に管理され、不正アクセスから保護されます。
- **スケーラビリティ**: クラスおよびメソッドは拡張可能であり、新しい機能追加に柔軟に対応可能です。

## 4. 技術要件
- **プログラミング言語**: Python 3.x
- **ライブラリ**: OpenAI API、logging、os、sys、argparse、traceback、configparser、typing、fnmatch
- **環境**: Python環境が構築されたOS上で動作
- **設定ファイル**: `settings.ini` でAPIキーやディレクトリ設定を管理

この仕様書は、アプリケーションの全体像と主要な機能を概説し、技術的な要件を明確にすることで、開発者が効率的かつ効果的にシステムを運用・保守できるようにすることを目的としています。
